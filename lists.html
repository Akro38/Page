List of useful tools in programming

Code editors and IDEs (Development Environments):

  • Visual Studio Code (VS Code) The most popular, lightweight editor with a huge number of plugins.
  • JetBrains IDEs: Specialized environments: IntelliJ IDEA (Java), PyCharm (Python), WebStorm (JS), PhpStorm (PHP).
  • Sublime Text: A very fast text editor.
  • Vim/Neovim Powerful editors for working in the terminal.

Version control and code hosting:

  1. Git: The standard for tracking changes to code.
  2. GitHub: The premier platform for storing code and collaborating.
  3. GitLab: An alternative with strong CI/CD tools.

Debugging and testing:

GDB GNU Debugger
Debugger for C/C++.
Postman
A convenient tool for API testing.
Valgrind
Memory and performance analysis.
JUnit/PyTest
Unit testing frameworks.
